# Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**.
This project showcases my skills, projects, and contact information in a clean and professional design.

---

##  Features

* Responsive design for mobile, tablet, and desktop
* Sticky navigation bar
* About section with skills and education
* Portfolio/projects section
* Contact form connected with Google Sheets
* Smooth scrolling navigation
* Social media links
* Modern UI with hover effects

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons
* Google Apps Script
* Google Sheets API
